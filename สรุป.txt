Conditionals and Control Flow
    Key concept
     If Statement: ทำการรันโค้ดบล็อกเมื่อเงื่อนไขที่ระบุเป็นจริง

     Elif Statement: อนุญาตให้ตรวจสอบเงื่อนไขหลาย ๆ เงื่อนไข หากเงื่อนไขก่อนหน้านั้นเป็นเท็จ

     Else Statement: ทำการรันโค้ดบล็อกเมื่อไม่มีเงื่อนไขใดที่เป็นจริง

     Single Condition: 'If condition:' มีแค่เงื่อนไขเดียว

     Multiple Condition: 'If condition1: elif condition2: else:' มีหลายเงื่อนไข
     
     Syntax: การเว้นวรรคและโครงสร้างที่ถูกต้องมีความสำคัญต่อความสามารถในการอ่านและการทำงาน

     Boolean: ตรรกะ

     Nested comdition: การซ้อนเงื่อนไข

     Flow Control: กำหนดเส้นทางการทำงานของโปรแกรม

     Default action: else จะทำงานเมื่อไม่มีเงื่อนไขใดเป็นจริง

     Clarity: ทำให้โค้ดอ่านง่ายขึ้น
Condition Statement
 If/Elif/Else
   โครงสร้างของ If/Elif/Else
      if condition:
         โค้ดที่จะทำงานเมื่อเงื่อนไข If เป็นจริง
      elif condition2:
         โค้ดที่จะทำงานเมื่อเงื่อนไขที่ 2 เป็นจริง
      else:
         โค้ดจะทำงานก็ต่อเมื่อเงื่อนไขด้านบนเป็นเท็จ

Comparision Operators

   Key concept
    Comparision Operators: ใข้ในเงื่อนไขการเปรียบเทียบตัวแปร

    Conditional Statement: ส่วนใหญ่ใข้ if/elif/else ในการควบคุมการทำงานของโปรแกรม

    Numeric and String Comparision: สามารถเปรียบเทียบตัวเลขและข้อความได้

    Decision Making: จำเป็นสำหรับการตัดสินใจและการรันโค้ดที่แตกต่างกันตามเงื่อนไข

   (>) มากกว่า จะเป็นจริงเมื่อค่าด้านซ้ายมากว่าด้านขวา
   ตัวอย่าง x > y

   (<) น้อยกว่า จะเป็นจริงเมื่อค่าด้านซ้ายน้อยว่าด้านขวา
   ตัวอย่าง x < y

   (==) เท่ากับ เป็นจริงเมื่อค่าของตัวแปรทั้งคู่เท่ากัน
   ตัวอย่าง x == y

   (!=) ไม่เท่กาับ เป้นจริงเมื่อค่าตัวแปรไม่เท่ากัน
   ตัวอย่าง x != y

   (>=) มากกว่าเท่ากับ เป็นจริงเมื่อค่าด้านซ้ายมากกว่าหรือเท่ากับกับค่าฝั่งขวา
   ตัวอย่าง x >= y

   (<=) น้อยกว่าเท่ากับ เป็นจริงเมื่อค่าด้านซ้ายน้อยกว่าหรือเท่ากับกับค่าฝั่งขวา
   ตัวอย่าง x >= y

   การเปรียบเทียบค่าของตัวแปรที่เป็นตัวเลขจะเห็นได้เลย 
   แต่การเปรียบเทียบข้อความ จะมีรหัสของตัวอักษรนั้นๆ ชื่อว่า รหัส ascii
